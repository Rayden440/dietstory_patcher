Step 1: Create a new file with the exact name ".env" in the root client folder (the same location as this readme).
Step 2: Open the file you've just created with the text editor of your choice.
Step 3: Copy the text below that is between the hypens (---) and paste into your .env file.
Step 4: Change the values to the correct ones for your server and save the file.

----------------------------------------------------------------------------------------
# APPLICATION SETTINGS
ELECTRON_MODE=dev

# PATCH SERVER DOMAIN AND PATHS
DS_WEB_SERVER_DOMAIN=127.0.0.1
DS_WEB_SERVER_PROTOCOL=https
DS_WEB_SERVER_PORT=443
DS_WEB_SERVER_HOME=https://127.0.0.1
----------------------------------------------------------------------------------------
